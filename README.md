# AA Aerospace Hackpad!
Custom macropad with a 3x2 matrix of switches with an OLED display!

## BOM
- 1 SEEEDUINO XIAO RP2040
- 6x Cherry MX Switches 
- 4x M3x5x4 Heatset inserts
- 4x M3x10mm Screws
- 1x 0.91" 128x32 OLED Display
- 1x Case
- 1x Top plate

## Build it!
Print the top plate and case files, insert the threaded inserts, drop the pcb in, and screw on the top. Use CircuitPython firmware and flash the main.py file and download the KMK firmware library.
